# 单表数据记录查询
## 一、简单数据查询
```sql
SELECT filed1 field2, ... ,fieldn FROM table_name
```

### 1. 查询所有字段数据
```sql
SELECT * FROM table_name;
```

也可以调整SELECT关键字后面的字段列表顺序，来设置字段的显示顺序。  
```sql
SELECT ename,job,MGR,Hiredate,sal,comm,deptno,emptno FROM table_name;
```
